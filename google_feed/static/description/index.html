Google Feed for Products

Google Feed for Product module used to display your products on google marketplace.Escape time-consuming and manual data feeds management.This module is automatically generat feeds according to the feed configurations.

Key Feature Of Google Feed for Products

Ready to use template

Product feeds easily generate with ready to use templates.

Conditions-based Attribute

Add condition-based attributes to your google feed feeds.Also add the optional attributes of feed

Automation

Set up the schedule for automatic and manually feed generation.

Increase Sales efficiency

Increase your sales efficiency with Google marketplace.

Multiple Files Types

You will be able to generate XML and TXT feed file.

Add Price List

Get to choose any of the pricelist to display product price.

View Feed File Using Link

With use of link you are able to see the generated feed file.

Feed Base On Product Type

With use of product type user can select product or product variant.

Send Feed To Remote Server

Using FTP and SFTP upload feed to remote server.

Enabled google feed menu

By default, the Google Feed menu will not be seen in the menu bar. For making it visible, Users->Technical Settings -> click on the checkbox Generate Google Feed

Feed Configuration

For generating google feed for the product, you can select the Main menu. Settings -> Google Feed -> Click on the Google Feed Configuration.

Feed file Configuration

Thereafter, create your new google feed and give Feed Name and File Name.

File Type Selection

Now, you will be able to select the file extension out of XML and TXT to generate your feed.

Add Price List

Now, The product price is displayed according to your selected price list. Inside generate google feed product price is added according to the selected price list

Set Product Type

Set the type of product, with use of this option you can change the type of product like Product , Variant

Field Mapping

At this step, you can defin basic attribute you want to output at the feed. Accurately specify which attribute values should chosen as a ID,Title,Description,Condition etc.

Product Configuration

Now, click on the ‘Feed configuration’ tab, you will get two options which are: “All Products” and “Exclude out of stock product”.

Here, by default, the checkbox ‘ALL Products’ is selected which will add all the products in google feed. However, you can also add your specific products by unchecking the checkbox ‘All Products’.

Add Server

If you want to upload the generated google feed onto your server, then you can follow these instructions. Here, you can see the ‘Server Configuration’ tab, under that, click on the checkbox Upload To server.

Server Details

Create or select the Server Name by filling the details. You can select your own FTP or SFTP server and then click on the Test connection button to check the server connection. Thereafter, enter your server folder name in Server Path.

Configuration Type

Inside Configuration Type you have two types of choice like Using File and Using Password
if you choose Using Password you have to give a password of server inside Password
and if you choose Using File you have to select key file of your server inside Server Configuration File

Auto Generate

If you want to auto-generate google feed, then click on the checkbox Auto Generate, which will enable the Cron schedule tab.

Error History

During the process of feed generation (automated or manual), if any error occurs then it will be shown in the Error History tab .

Download Feed File

For downloading the generated feed file, click on the Generated File option.

Feed Url Link

To see the genegrated feed file using browser,click on theFeed URL option.

Google Merchant Configuration

Using this module Google Feed for Product generate the feed file, after you want to send this file to SFTP or FTP for further google process. For this process you need to create a Google Merchant Account. So below step through you can create and configure google merchant account.

Sign up for Google Merchant Center

For Sign up the Google Merchant account go to the https://www.google.com/retail/solutions/merchant-center/.

Add Generated Feed in Google Merchant Account

After successfully signup and set the basic information in google merchant account. Go to the Producst -> Feeds menu.

Add Basic Information

Now, You can see the previously updated feed and you can also add the new feed.Press the plus button for add the new feed. You can see the below basic information screen. Now set the country,language and other information and press next.

File Selection

Now, You can add the file name and input method. choose imput method upload and press continue for next step.

Setup

Now, enter feed file name and browse or drag and drop the generated feed file. If you want to upload the test file at that time you need to tick the Upload to Test. After press the Create Feed Button.

Is this app compatible with Odoo Enterprise as well as Community?

Yes, the app works perfectly well with Odoo Enterprise (On-premise and Odoo.SH) as well as Community. Odoo Online (Cloud) does not allow installation of third-party apps and hence this app cannot be installed on Odoo Online.
