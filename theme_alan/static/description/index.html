Why Choose Theme Alan?

Alan is one of the most powerful, amazing, and flexible theme in Odoo store. Alan is the professional and multipurpose bootstrap base theme for any business and website. The code is very well organized and modular so you can build any layout with your imagination.

Some salient features of Alan are flexible design, advanced mega menu, drag and drop snippets, HTML builder block, amazing header footer options, beautiful shop details page, shop full-width options with the list and grid view, optimized responsive behavior, infinite product loader, Product Quick Overview.

Yes! You can install this theme in just one-click. Theme installation is very simple.

Login into your website and just install the Atharva Theme General module and Go to Website --> Configuration --> Settings. Select the Website and then click on the "Choose a Theme" button. You will be redirected to the available themes. and from there you can use "Theme Alan".
